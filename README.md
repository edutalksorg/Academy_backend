# College Placement Backend

Simple Express + Sequelize backend scaffold for the College Placement & Student Management platform.

Quick start

1. Copy `.env.example` to `.env` and adjust DB credentials.
2. Install dependencies:

```powershell
cd backend
npm install
```

3. Run in development:

```powershell
npm run dev
```

4. To run seeder (creates SuperAdmin and sample data):

```powershell
npm run seed
```

API: Basic auth routes available under `/api/auth` (`/register`, `/login`).
